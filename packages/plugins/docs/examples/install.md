## 安装

::: code-group

```bash [npm]
npm install @wgl/plugins -S
```

```bash [cnpm]
cnpm install @wgl/plugins -S
```

```bash [pnpm]
pnpm add @wgl/plugins
```

```bash [yarn]
yarn add @wgl/plugins
```

:::

## 注意

`@wgl/plugins` **仅适用于 Browser 环境**，请勿在 Node.js 中引用。

`upload` 插件可选依赖 `axios`，按需安装：

```bash
pnpm add axios
```
