# 04 - 目录结构

## 迁移后 monorepo 文档相关目录

```
wgl-monorepo/
├── docs/
│   ├── .vitepress/
│   │   └── config.js              ← 唯一 VitePress 配置
│   ├── index.md                   ← 全站首页（@wgl-m/* 总览）
│   │
│   ├── packages/                  ← 对外文档内容（纳入站点）
│   │   ├── utils/
│   │   │   ├── index.md           ← 包首页（layout: home）
│   │   │   └── examples/
│   │   │       ├── index.md
│   │   │       ├── install.md
│   │   │       ├── debounce.md
│   │   │       └── ...
│   │   ├── plugins/
│   │   │   ├── index.md
│   │   │   └── examples/
│   │   │       └── ...
│   │   ├── node-utils/
│   │   │   ├── index.md
│   │   │   └── install.md
│   │   └── css/
│   │       ├── index.md
│   │       └── usage.md           ← reset / mixin / bem 说明
│   │
│   ├── guides/                    ← 可选：发版、迁移摘要
│   │   └── publishing.md
│   │
│   ├── typescript-migration/      ← 内部规划，srcExclude
│   ├── documentation-site/        ← 本方案，srcExclude
│   └── README.md                  ← 仓库内文档索引，srcExclude
│
├── packages/
│   ├── utils/
│   │   └── src/                   ← @include 目标，不再含 docs/
│   ├── plugins/
│   │   └── src/
│   ├── node-utils/
│   └── css/
│
├── scripts/
│   └── deploy-docs.sh             ← 统一部署脚本（可选）
│
└── package.json                   ← docs / docs:build 脚本
```

## 删除 / 废弃

```
packages/utils/docs/          → 内容迁至 docs/packages/utils/，整目录删除
packages/plugins/docs/        → 内容迁至 docs/packages/plugins/，整目录删除
packages/utils/docs/package.json
packages/plugins/docs/package.json
packages/*/docs/deploy.sh
packages/*/docs/.vitepress/
```

## URL 映射

假设 `base: '/wgl-monorepo/'`，则：

| 旧 URL | 新 URL |
|--------|--------|
| `.../page__package-utils/examples/debounce` | `.../wgl-monorepo/packages/utils/examples/debounce` |
| `.../page__package-plugins/examples/clickOutside` | `.../wgl-monorepo/packages/plugins/examples/clickOutside` |
| `.../page__package-utils/` | `.../wgl-monorepo/packages/utils/` 或全站 `/` |

## 首页 `docs/index.md` 草案

```yaml
---
layout: home
hero:
  name: "@wgl-m/*"
  text: "WGL Monorepo"
  tagline: Utils · Plugins · Node Utils · CSS
  actions:
    - theme: brand
      text: Utils
      link: /packages/utils/
    - theme: alt
      text: Plugins
      link: /packages/plugins/
features:
  - title: 统一文档
    details: 单一 VitePress 站点，本地 pnpm docs 预览
  - title: TypeScript
    details: 全包类型导出，IDE 友好
---
```

## workspace 变更

**pnpm-workspace.yaml**（迁移后）：

```yaml
packages:
  - 'packages/*'
```

移除 `'packages/*/docs'`。
