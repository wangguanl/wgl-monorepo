## @wgl-m/node-utils

### 迁移前

```
packages/node-utils/
├── index.js          ← 源码即产物
├── package.json
└── .npmrc
```

### 迁移后

```
packages/node-utils/
├── src/
│   └── index.ts
├── dist/
│   ├── index.js
```

## @wgl-m/plugins

### 变更点

| 项 | 变更 |
|----|------|
| 源文件 | `.js` → `.ts` |
| tsup | 加 `dts: true`，entry glob 改 `src/*.ts` |
| tsconfig | 新增，`lib: ["ES2020", "DOM"]` |
| exports | 每个子路径加 `types` 字段 |

### tsup.config.ts 预览

```ts
import { defineConfig } from 'tsup';
import { globSync } from 'glob';

const entries = globSync('src/*.ts').reduce((acc, file) => {
  acc[file.replace(/^src\//, '').replace(/\.ts$/, '')] = file;
  return acc;
}, {} as Record<string, string>);

export default defineConfig({
  entry: entries,
  format: ['esm', 'cjs'],
  dts: true,
  external: ['axios'],
  outExtension({ format }) {
    return { js: format === 'cjs' ? '.cjs' : '.js' };
  },
  clean: true,
  sourcemap: true,
  outDir: 'dist',
});
```

### 特殊文件

| 文件 | 处理 |
|------|------|
| `upload.ts` | `axios` 保持 external；`window.baseUrl` 声明全局类型或改用参数传入 |
| `scale.ts` | 访问 `document`，需 DOM lib |
| `toggleFullscreen.ts` | `el: string` 参数类型为 CSS selector |

## @wgl-m/utils

### 变更点（最大）

| 项 | 变更 |
|----|------|
| 构建 | Webpack 5 → tsup |
| 源文件 | 15 个 `.js` → `.ts` |
| 删除 | `scripts/`、`babel.config.js`、Webpack deps |
| 输出 | 新增 `.d.ts`，移除 ES5 UMD |

### tsup 多 entry 策略

```ts
// 与现有 Webpack esmodule.conf 行为对齐
const entries = globSync('src/**/*.ts').reduce((acc, file) => {
  const name = file.replace(/^src\//, '').replace(/\.ts$/, '');
  acc[name === 'index' ? 'index' : `es/${name}`] = file;
  return acc;
}, {} as Record<string, string>);
```

### exports 预览

```json
{
  "exports": {
    ".": {
      "import": {
        "types": "./dist/index.d.ts",
        "default": "./dist/index.js"
      },
      "require": {
        "types": "./dist/index.d.cts",
        "default": "./dist/index.cjs"
      }
    },
    "./es/*": {
      "import": {
        "types": "./dist/es/*.d.ts",
        "default": "./dist/es/*.js"
      }
    }
  }
}
```

> 注：tsup 的 es 分包 `.d.ts` 路径需实际构建后验证，可能需要 `dts` 配置 `entry` 与 `format` 联调。

### 各文件类型策略

| 文件 | 关键类型 |
|------|----------|
| `deepCopy.ts` | `<T>(obj: T): T` |
| `deepMerge.ts` | `<T extends Record<string, unknown>>(target: T, ...sources: Partial<T>[]): T` |
| `debounce.ts` | `(fn: () => void, delay?: number) => void` |
| `formatDate.ts` | 导出多个格式化函数，参数 `Date \| string \| number` |
| `parseUrl.ts` | 返回 `{ [key: string]: string }` 或专用 interface |
| `validation.ts` | 规则对象 interface |
| `unique.ts` | `<T>(arr: T[]) => T[]` |
| `rem.ts` | px 换算，参数 `number` |
| `hump.ts` | 字符串转换 `(str: string) => string` |
| `isHtmlTag.ts` | `(tag: string) => boolean` |
| `isDateValid.ts` | `(date: unknown) => boolean` |
| `isArrayNested.ts` | `(arr: unknown) => boolean` |
| `arrDiff.ts` | 数组差集 |
| `recursionSegments.ts` | 递归分段 |
| `verifyVar.ts` | 变量校验 |
