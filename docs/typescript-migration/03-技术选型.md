# 03 - 技术选型

## TypeScript

| 项 | 选择 | 说明 |
|----|------|------|
| 版本 | `^5.7` | 与各包已有 devDep 一致 |
| 配置位置 | 根 `tsconfig.base.json` + 各包 `tsconfig.json` | monorepo 标准做法 |
| 初始 strict | `false` | 降低迁移阻力 |
| target | `ES2020` | 覆盖 Node 16+ 与现代 Browser |
| module | `ESNext` | 交给 tsup 处理输出格式 |
| 类型检查 | `tsc --noEmit` | 只做检查，不直接编译 |

### 根 tsconfig.base.json（预览）

```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "declaration": true,
    "declarationMap": true,
    "sourceMap": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "strict": false,
    "resolveJsonModule": true,
    "isolatedModules": true
  }
}
```

## 打包工具：统一 tsup

| 包 | 现在 | 迁移后 |
|----|------|--------|
| utils | Webpack 5 + Babel | **tsup** |
| plugins | tsup | tsup（加 `dts: true`） |
| node-utils | 无 | **tsup** |
| css | Gulp | 不变 |

### 为什么 utils 放弃 Webpack？

- Webpack 配置 4 个文件 + Babel，维护成本高
- tsup 原生支持 ESM + CJS + `.d.ts` + sourcemap
- plugins 已验证 tsup 可行

### utils 输出格式变化

| 现在 | 迁移后 | 说明 |
|------|--------|------|
| `main.mjs` | `index.js` (ESM) | exports 字段同步更新 |
| `main.cjs` | `index.cjs` (CJS) | 同上 |
| `main.js` (ES5 UMD) | **移除** | 见 Breaking Changes |
| `es/*.mjs` | 保留分包 | tsup 多 entry |

### tsup 公共配置

```ts
// 示意
export default defineConfig({
  entry: ['src/index.ts'],
  format: ['esm', 'cjs'],
  dts: true,
  sourcemap: true,
  clean: true,
  splitting: false,
});
```

## 类型导出

每个包的 `package.json` 增加：

```json
{
  "types": "./dist/index.d.ts",
  "exports": {
    ".": {
      "import": {
        "types": "./dist/index.d.ts",
        "default": "./dist/index.js"
      },
      "require": {
        "types": "./dist/index.d.cts",
        "default": "./dist/index.cjs"
      }
    }
  }
}
```

## 共享 tooling 包（可选）

```
tooling/
  tsconfig/
    base.json
    node.json      # node-utils 专用
    browser.json   # plugins 专用
    universal.json # utils 专用（Node + Browser）
```

初期可直接根目录 `tsconfig.base.json`，不单独建 tooling 包。

## 依赖调整

### 根 package.json 新增

```json
{
  "scripts": {
    "typecheck": "turbo run typecheck"
  },
  "devDependencies": {
    "typescript": "^5.7.2"
  }
}
```

### 各包新增

```json
{
  "scripts": {
    "typecheck": "tsc --noEmit"
  },
  "devDependencies": {
    "typescript": "^5.7.2",
    "tsup": "^8.3.5"
  }
}
```

### utils 可移除

```
webpack, webpack-merge, babel-loader, @babel/*, terser-webpack-plugin, clean-webpack-plugin
```

## CI 更新

`release.yml` 无需改动（仍跑 `pnpm release` → `pnpm build`）。

可选在 PR 时加 `typecheck` job（Phase 5）。
